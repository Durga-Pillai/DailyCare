# DailyCare
A frontend taskboard for dialysis center staff to manage recurring 
and ad-hoc patient care tasks across multiple roles and shifts
---

## Purpose

Dialysis teams manage complex, recurring tasks per patient — monthly 
labs, access checks, diet counselling, vaccination reminders, social 
work follow-ups and more. Multiple roles (nurse, dietician, social 
worker, technician, doctor) coordinate around the same patient.

This frontend provides a clear, resilient interface to track tasks 
by status, role, and patient — built to handle real-world conditions 
like slow networks, missing data, and evolving requirements.

---

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- npm

---

## Getting Started

git clone https://github.com/Durga-Pillai/DailyCare.git
cd DailyCare
npm install
npm run dev
